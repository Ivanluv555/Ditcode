# 接口契约总表（前后端统一）

## 1. 健康检查

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/health` | 服务健康状态 |

## 2. 认证模块

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/api/auth/default-login` | 默认账号自动登录 |
| GET | `/api/auth/session` | 获取当前会话用户 |
| POST | `/api/auth/register` | 注册 |
| POST | `/api/auth/login` | 登录 |
| POST | `/api/auth/logout` | 登出 |

## 3. 工作区模块

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/workspace` | 获取当前用户工作区聚合 |
| PUT | `/api/workspace` | 全量写回工作区聚合 |
| POST | `/api/workspace/reset` | 重置当前用户工作区 |

## 4. 社区模块

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/community` | 获取社区已发布档案 |
| POST | `/api/community/publish` | 发布档案 |
| POST | `/api/community/unpublish` | 取消发布 |
| POST | `/api/community/remix` | remix 计数 +1 |

## 5. 模型网关模块（新增）

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/api/model/generate` | 前端生成请求 -> 后端整理 -> 上游模型 |

### 请求体（示例）

```json
{
  "taskId": "task_1710000000000",
  "archiveId": "archive_1710000000000",
  "prompt": "现代风格咖啡店",
  "imageName": "ref.png",
  "imagePreview": "data:image/png;base64,..."
}
```

### 响应体（示例）

```json
{
  "ok": true,
  "provider": "upstream",
  "taskId": "task_1710000000000",
  "prompt": "现代风格咖啡店",
  "imagePreview": "https://...",
  "finishedAt": 1710000001234,
  "raw": {}
}
```

## 6. 错误响应规范

统一返回：

```json
{
  "ok": false,
  "message": "错误描述"
}
```
