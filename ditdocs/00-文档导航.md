# Ditcode 文档导航

本目录用于统一维护从项目起始到当前阶段的完整技术资料。

## 文档列表

1. `00-文档导航.md`：文档总览（本文件）
2. `01-项目演进与里程碑.md`：从 mock 到 SpringBoot 的阶段演进记录
3. `02-前端技术选型与实现.md`：前端架构、技术选型、状态与交互设计
4. `03-后端架构与模块设计.md`：后端 core/module 分层、鉴权、模型网关设计
5. `04-数据库设计与规范.md`：关系模型、约束规则、code-first 策略
6. `05-接口契约总表.md`：前后端 API 契约汇总（含模型转发接口）
7. `业务用例表.md`：业务用例、规则、验收建议
8. `SpringBoot后端说明.md`：后端快速接入与运行说明

## 外部关联文档

- `ditdbase/数据库设计.md`：数据库全量设计说明（详细版）
- `ditdbase/db/schema/*.sql`：规范化 SQL 草案与约束参考

## 当前统一约定（强制）

1. 图标：前端统一使用 **Iconify**，后端不参与图标资源管理。
2. 资源边界：前端负责静态资源；后端仅负责动态资源地址与模型结果。
3. 后端结构：全局能力放 `core`，业务域放 `module`。
4. 持久化：统一使用 JPA，`spring.jpa.hibernate.ddl-auto=update`（code-first）。
