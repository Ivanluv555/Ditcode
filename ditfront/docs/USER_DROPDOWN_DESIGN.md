# 个人信息下拉栏设计文档

## 设计概述

个人中心下拉菜单采用三段式垂直布局，固定在页面右上角头像按钮下方。

## 结构说明

### 1. 顶部用户信息区
- **圆形头像**：64px 直径，渐变背景（紫色系），居中显示
- **用户昵称**：16px 字号，加粗，居中对齐
- **邮箱地址**：13px 字号，次要文字颜色，居中对齐

### 2. 中段主题设置区
- **标题**："外观主题"，13px 字号，加粗
- **切换组件**：横向两个按钮
  - 亮色主题：太阳图标 + "亮色"文字
  - 深色主题：月亮图标 + "深色"文字
  - 激活状态：主题色背景，白色文字

### 3. 底部功能列表区
垂直排列的功能入口，每项包含图标和文字：
- 历史记录（时钟图标）
- 使用通知（铃铛图标）
- 切换账号（左右箭头图标）
- **联系我们（信封图标）** - 点击后打开联系我们弹窗
- **意见反馈（对话气泡图标）** - 点击后打开意见反馈弹窗
- 注销账号（退出图标，红色危险样式）

## 新增功能弹窗

### 联系我们弹窗 (ContactModal.vue)
**位置**：`/src/shared/components/ContactModal.vue`

**功能**：
- 显示多种联系方式（邮箱、微信、QQ群、客服热线）
- 每个联系方式包含图标、标签、联系信息和描述
- 提供复制按钮，一键复制联系信息
- 显示工作时间和响应时间说明
- 返回按钮可回到用户信息弹窗

**布局**：
- 顶部：返回按钮 + 标题
- 中部：联系方式列表（4种方式）
- 底部：工作时间和响应时间提示

### 意见反馈弹窗 (FeedbackModal.vue)
**位置**：`/src/shared/components/FeedbackModal.vue`

**功能**：
- 反馈类型选择（问题反馈、功能建议、体验优化、其他）
- 详细描述输入框（必填）
- 联系方式输入框（可选）
- 表单验证和提交
- 返回按钮可回到用户信息弹窗

**布局**：
- 顶部：返回按钮 + 标题
- 中部：反馈表单（类型选择 + 描述 + 联系方式）
- 底部：取消和提交按钮

## 动画效果

使用 GSAP 实现：
- 整体面板：从右上角缩放淡入（scale + opacity）
- 列表项：依次从右侧滑入（stagger 动画，间隔 56ms）
- 退出动画：缩放淡出

## 交互行为

- 点击头像按钮：显示/隐藏下拉菜单
- 点击遮罩层：关闭下拉菜单
- 点击功能项：执行对应操作并关闭菜单
- 主题切换：立即生效，菜单保持打开
- **点击"联系我们"**：关闭用户信息弹窗，打开联系我们弹窗
- **点击"意见反馈"**：关闭用户信息弹窗，打开意见反馈弹窗
- **弹窗内点击返回按钮**：关闭当前弹窗，返回用户信息弹窗
- **弹窗内点击遮罩层**：直接关闭当前弹窗

## 样式特性

- 宽度：320px
- 圆角：12px
- 阴影：elevated 级别
- 分割线：1px 细线，使用边框颜色
- 响应式：自动适配深色/亮色主题

## 实现文件

- `/src/App.vue` - 主要实现代码，包含用户信息弹窗和弹窗状态管理
- `/src/shared/components/ContactModal.vue` - 联系我们弹窗组件
- `/src/shared/components/FeedbackModal.vue` - 意见反馈弹窗组件
- `/src/shared/styles/global.css` - 全局CSS变量定义
- 使用 `@iconify/vue` 提供图标
- 使用 GSAP 提供动画
- 使用 CSS 变量支持主题切换

## 技术要点

1. **组件复用**：新弹窗使用与用户信息弹窗相同的尺寸（320px）和样式系统
2. **状态管理**：使用独立的 ref 变量管理每个弹窗的显示状态
3. **导航流程**：用户信息弹窗 → 联系我们/意见反馈弹窗 → 返回用户信息弹窗
4. **动画一致性**：所有弹窗使用相同的 GSAP 动画效果
5. **主题适配**：所有组件使用 CSS 变量，自动适配亮色/深色主题
